# MaxPlus 网站使用手册

## 1. 手册说明

本文档用于指导用户使用 MaxPlus 网站，包括：

- 前台官网使用：面向访客、采购客户、合作客户。
- 后台管理使用：面向运营人员、管理员、客服人员。

本文档不涉及代码、服务器部署和数据库维护。正式网站地址、后台地址、管理员账号由网站负责人或系统管理员提供。

## 2. 用户角色

| 角色 | 使用范围 | 主要操作 |
| --- | --- | --- |
| 网站访客 | 前台官网 | 浏览首页、商品、视频、博客、品牌故事、政策页面 |
| 采购客户 | 前台官网 | 搜索商品、查看商品详情、提交批发询价或联系表单 |
| 运营人员 | 后台管理 | 维护首页、商品、视频、博客、品牌内容、SEO |
| 客服人员 | 后台管理 | 查看询盘、处理订阅、更新询盘状态 |
| 管理员 | 后台管理 | 登录后台、维护系统设置、邮件配置、查看日志 |

## 3. 前台官网使用手册

### 3.1 访问官网

打开浏览器，输入官网地址进入 MaxPlus 网站。

常见入口：

- 首页
- 商品列表
- 视频列表
- 博客列表
- 品牌故事
- 联系我们
- 购买入口

网站顶部导航栏提供主要页面入口。移动端访问时，点击菜单按钮展开导航。

### 3.2 首页使用

首页用于快速了解 MaxPlus 品牌、重点商品、视频内容和联系入口。

首页常见模块：

| 模块 | 用途 |
| --- | --- |
| 顶部导航 | 进入商品、视频、博客、品牌故事、联系页面 |
| 首屏轮播 | 展示品牌主视觉、重点卖点和快捷按钮 |
| 视频模块 | 展示推荐视频，可进入视频详情 |
| 精选商品 | 展示重点推荐商品，可进入商品详情 |
| 分类模块 | 展示商品分类入口 |
| 评价模块 | 展示客户评价、评分和反馈 |
| 联系模块 | 提交合作、采购或业务咨询 |
| 订阅弹窗 | 提交邮箱或其他信息，订阅最新资讯 |
| 返回顶部按钮 | 页面滚动较深时快速回到顶部 |

操作方法：

1. 点击顶部导航进入对应页面。
2. 点击首页商品卡片进入商品详情。
3. 点击视频卡片进入视频详情。
4. 在联系模块填写表单并提交。
5. 如出现订阅弹窗，可填写信息订阅，也可关闭。

### 3.3 浏览商品

进入“Products”商品页面后，可以查看全部商品。

可用操作：

- 按分类筛选商品。
- 按排序方式调整商品顺序。
- 点击商品卡片查看详情。
- 点击分类卡片进入分类详情页。
- 点击“Search Catalog”进入搜索页。

商品排序选项通常包括：

- Featured：优先显示精选商品。
- Latest：显示最新商品。
- Price Low To High：价格从低到高。
- Price High To Low：价格从高到低。

### 3.4 查看商品详情

在商品列表或分类页点击商品卡片后，进入商品详情页。

商品详情页可查看：

- 商品名称
- 商品分类
- 商品图片或图库
- 商品摘要
- 价格信息
- 商品描述
- 购买或询价按钮
- 相关商品推荐

图片操作：

- 如果商品有多张图片，可点击缩略图切换主图。
- 在桌面端使用鼠标悬停图片时，部分商品图片支持局部放大查看。

购买或询价：

- 点击商品页中的购买按钮，可进入购买入口页或外部购买链接。
- 如果需要批量采购，建议进入“Go To Buy”页面提交询价。

### 3.5 使用商品分类页

点击商品分类后进入分类详情页。

分类页可用于查看某一类商品，并进一步筛选。

常见筛选项：

- Sport Type：运动类型。
- Audience：适用人群。
- Use：使用场景。
- Stock：库存状态。
- Sort：排序方式。

操作方法：

1. 选择筛选条件。
2. 页面会自动显示符合条件的商品。
3. 如果商品较多，可使用分页按钮切换页码。
4. 点击商品卡片进入商品详情。

### 3.6 使用搜索功能

进入“Search”页面后，可以搜索商品、分类和博客文章。

操作方法：

1. 在搜索框输入关键词。
2. 点击 Search 按钮。
3. 查看分类结果、商品结果和文章结果。
4. 点击结果进入对应页面。

如果没有输入关键词，页面会显示热门关键词，可直接点击查看。

如果没有搜索结果，可尝试：

- 使用更短的关键词。
- 使用商品分类名称。
- 使用运动类型、用途、品牌相关词。

### 3.7 浏览视频

进入“Videos”页面后，可以查看 MaxPlus 视频内容。

可用操作：

- 按视频大类筛选。
- 按子分类筛选。
- 点击视频卡片进入详情页。

视频卡片通常包含：

- 视频封面
- 分类
- 主题标签
- 标题
- 摘要

### 3.8 播放视频

进入视频详情页后，可以播放视频。

视频详情页可查看：

- 视频播放器
- 视频标题
- 视频分类
- 视频主题
- 视频说明
- 相关视频推荐

如果视频无法播放：

- 检查网络连接。
- 刷新页面后重试。
- 如果仍无法播放，可能是视频链接暂不可用，请联系网站管理员。

### 3.9 浏览博客文章

进入“Blog”页面后，可以查看文章列表。

可用操作：

- 按文章分类筛选。
- 输入关键词搜索文章。
- 按最新或最旧排序。
- 使用分页按钮切换文章列表。
- 点击文章进入详情页。

文章列表通常展示：

- 文章分类
- 标题
- 摘要
- 作者
- 发布时间
- 预计阅读时间

### 3.10 阅读博客详情

进入文章详情页后，可以阅读完整内容。

文章详情页包含：

- 文章标题
- 分类
- 作者
- 发布时间
- 阅读时间
- 正文内容
- 文章目录
- 相关文章推荐

如果文章右侧出现目录，可点击目录项快速跳转到对应段落。

### 3.11 查看品牌故事

进入“Brand Story”页面，可以了解 MaxPlus 品牌信息。

页面内容通常包括：

- 品牌介绍
- 品牌故事
- 关键数据
- 品牌亮点
- 商品分类入口

点击分类入口可跳转到对应商品分类页。

### 3.12 联系我们

进入“Contact”页面，可以查看联系方式并提交咨询。

页面提供：

- 销售邮箱
- 客服邮箱
- 电话
- 社交媒体链接
- 批发合作表单
- 普通业务咨询表单

提交表单步骤：

1. 填写姓名。
2. 填写邮箱。
3. 填写公司或组织名称。
4. 输入咨询内容。
5. 点击提交按钮。
6. 页面出现提交成功提示后，等待工作人员联系。

填写建议：

- 批发采购请写明目标产品、数量、采购周期。
- 合作咨询请写明公司名称、合作方向和联系方式。
- 售后或支持问题请尽量描述问题细节。

### 3.13 使用购买入口

进入“Go To Buy”页面，可以提交批发询价。

该页面适合：

- 批量采购客户
- 俱乐部、学校、团队采购
- 零售商或渠道客户
- 想快速了解推荐采购商品的用户

操作方法：

1. 查看页面上的采购说明。
2. 填写批发询价表单。
3. 填写想采购的产品、数量、时间安排等信息。
4. 点击提交。
5. 提交成功后等待工作人员跟进。

### 3.14 订阅资讯

网站可能显示订阅按钮或订阅弹窗。

操作方法：

1. 点击订阅按钮。
2. 填写弹窗中的必填信息。
3. 点击提交。
4. 出现成功提示后表示已提交。

订阅信息可能用于：

- 接收新品信息。
- 接收活动通知。
- 接收采购或业务相关信息。

### 3.15 Cookie 偏好设置

首次访问网站时，可能出现 Cookie 提示。

可选操作：

- Accept All：接受全部 Cookie。
- Preferences：打开偏好设置。
- Save Preferences：保存选择。

Cookie 类型说明：

| 类型 | 说明 |
| --- | --- |
| Essential | 基础功能所需，默认开启 |
| Analytics | 用于统计页面访问和内容表现 |
| Marketing | 用于后续营销或个性化跟进 |

### 3.16 查看政策页面

页脚通常包含政策链接，例如：

- Privacy Policy
- Shipping Policy
- Refund Policy
- Terms

点击对应链接后，可以查看详细政策说明。

### 3.17 前台常见问题

#### 搜索不到商品怎么办？

可以尝试更短的关键词，或从商品分类进入查找。

#### 表单提交失败怎么办？

检查姓名、邮箱、公司、留言等必填项是否填写完整。若仍失败，刷新页面后重试。

#### 商品价格是否为最终采购价？

页面价格通常用于参考。批量采购、渠道合作或定制需求，请通过“Go To Buy”或“Contact”提交询价确认。

#### 视频打不开怎么办？

可能是网络或外部视频平台问题。请刷新页面或稍后再试。

#### 如何联系工作人员？

可通过“Contact”页面提交表单，也可使用页面展示的邮箱或电话联系。

## 4. 后台管理使用手册

### 4.1 访问后台

打开浏览器，输入后台地址。

后台地址通常由管理员提供。进入后台后，系统会根据状态跳转：

- 未安装：进入安装页面。
- 已安装但未登录：进入登录页面。
- 已登录：进入仪表盘。

### 4.2 首次安装

首次使用后台时，需要完成安装。

安装页面需要填写：

- 数据库主机
- 数据库端口
- 数据库用户名
- 数据库密码
- 数据库名称
- 管理员账号
- 管理员密码
- 确认管理员密码

操作步骤：

1. 打开后台安装页面。
2. 填写数据库信息。
3. 点击数据库测试。
4. 测试成功后填写管理员账号和密码。
5. 点击安装。
6. 安装完成后进入登录页面。

注意事项：

- 管理员账号建议使用英文、数字、点、下划线或短横线。
- 管理员密码至少 8 位。
- 数据库账号需要具备创建数据库、创建表和写入数据的权限。

### 4.3 登录后台

操作步骤：

1. 打开后台登录页面。
2. 输入用户名。
3. 输入密码。
4. 选择角色。
5. 点击登录。

登录成功后进入后台仪表盘。

如果登录失败：

- 检查账号和密码是否正确。
- 检查是否选择了正确角色。
- 多次失败后可能触发登录限制，请稍后再试。

### 4.4 忘记密码

如果忘记后台密码，可以进入密码恢复页面。

操作步骤：

1. 点击或访问密码恢复页面。
2. 输入恢复密钥。
3. 输入账号。
4. 设置新密码。
5. 确认新密码。
6. 提交后重新登录。

恢复密钥由管理员保管，不应公开给普通运营人员。

### 4.5 后台界面说明

后台主要区域：

| 区域 | 说明 |
| --- | --- |
| 左侧菜单 | 进入各功能模块 |
| 顶部栏 | 显示当前页面标题、面包屑、当前用户和退出按钮 |
| 页面标签 | 显示已打开的后台页面，可切换或关闭 |
| 内容区 | 当前功能页面的操作区域 |

左侧菜单分组：

- 总览
- 首页运营
- 商品中心
- 视频管理
- 内容管理
- 博客管理
- 线索管理
- 系统配置

### 4.6 仪表盘

路径：

```text
总览 -> 仪表盘
```

作用：

- 查看网站内容和业务数据概览。
- 快速了解商品、视频、文章、询盘、订阅等数据情况。

适用人员：

- 管理员
- 运营人员
- 客服负责人

### 4.7 站点设置

路径：

```text
系统配置 -> 站点设置
```

可维护内容：

- 品牌名称
- Logo 相关文案
- 销售邮箱
- 客服邮箱
- 电话
- 社交媒体链接
- 页脚信息
- 网站主题
- 通知信息

操作步骤：

1. 进入站点设置页面。
2. 修改需要调整的字段。
3. 点击保存。
4. 刷新前台页面查看效果。

影响范围：

- 前台顶部导航。
- 前台页脚。
- 联系页联系方式。
- 网站主题和节日效果。

### 4.8 邮件配置

路径：

```text
系统配置 -> 邮件配置
```

用途：

- 配置订阅确认邮件。
- 配置发件人信息。
- 配置邮件模板。

填写建议：

- SMTP 主机、端口、账号、密码需要向邮箱服务商确认。
- 发件人邮箱应与 SMTP 账号匹配。
- 邮件模板建议保留品牌名称、用户提交信息等必要变量。

保存后，可通过前台订阅功能验证邮件是否正常发送。

### 4.9 SEO 设置

路径：

```text
系统配置 -> SEO 设置
```

可维护内容：

- 全局标题
- 全局描述
- Open Graph 图片
- robots 是否允许索引
- sitemap 是否启用
- 商品标题模板
- 分类标题模板
- 博客标题模板

适用场景：

- 调整搜索引擎展示标题。
- 优化页面描述。
- 控制搜索引擎收录策略。

修改后建议检查前台页面标题和描述是否符合预期。

### 4.10 系统日志

路径：

```text
系统配置 -> 系统日志
```

作用：

- 查看登录记录。
- 查看内容保存、删除等操作记录。
- 查看前台询盘和订阅产生的相关日志。

可按日志类型筛选。

## 5. 首页运营操作

### 5.1 首页轮播

路径：

```text
首页运营 -> 首页轮播
```

用途：

- 管理首页首屏轮播。
- 设置轮播标题、副标题、按钮、跳转链接和背景图。

操作建议：

- 标题保持简洁。
- 按钮链接应指向商品、分类、视频或购买页。
- 图片建议使用清晰、横向视觉图。
- 不需要展示的轮播项可以关闭或删除。

前台影响：

- 首页首屏区域。

### 5.2 首页产品

路径：

```text
首页运营 -> 首页产品
```

用途：

- 设置首页精选商品模块标题、说明和按钮文案。
- 选择首页展示的商品。

操作步骤：

1. 设置模块标题和说明。
2. 选择要展示的商品。
3. 点击保存。
4. 前台首页刷新后查看展示效果。

注意：

- 商品需要先在“商品管理”中创建。
- 已下架或未发布商品不建议放到首页。

### 5.3 首页视频

路径：

```text
首页运营 -> 首页视频
```

用途：

- 设置首页视频模块文案。
- 选择首页主推视频。

注意：

- 视频需要先在“视频管理”中创建。
- 视频链接应确保可播放。

### 5.4 首页分类

路径：

```text
首页运营 -> 首页分类
```

用途：

- 设置首页分类模块标题和说明。
- 管理首页展示的分类入口。

注意：

- 分类需要先在“商品分类”中创建。
- 首页通常只展示重点分类，避免入口过多。

### 5.5 首页评价配置

路径：

```text
首页运营 -> 首页评价配置
```

用途：

- 设置评价模块标题。
- 设置评价摘要。
- 设置评分或统计指标。
- 设置评价展示方式。

前台影响：

- 首页客户评价模块。

### 5.6 评论内容管理

路径：

```text
首页运营 -> 评论内容管理
```

用途：

- 新增、编辑、删除具体评价内容。
- 设置评价作者、评分、说明、头像等信息。

建议：

- 评价内容应真实、简洁。
- 星级评分保持一致口径。
- 图片应清晰，不要使用过大文件。

### 5.7 首页联系

路径：

```text
首页运营 -> 首页联系
```

用途：

- 设置首页联系模块标题和说明。
- 设置提交成功提示。
- 配置前台联系表单字段。

可配置字段类型：

- 文本
- 邮箱
- 电话
- 下拉选择
- 多行文本

字段可设置：

- 是否启用
- 是否必填
- 占位提示
- 下拉选项

保存后，前台首页联系表单会同步变化。

### 5.8 订阅弹窗

路径：

```text
首页运营 -> 订阅弹窗
```

用途：

## 11. 后台常见问题

### 11.1 保存后前台没有变化

可尝试：

1. 确认后台已点击保存。
2. 刷新前台页面。
3. 检查该内容是否启用。
4. 检查首页模块开关是否打开。
5. 检查商品、视频、文章状态是否允许展示。

### 11.2 商品详情页打不开

可能原因：

- 商品 slug 填写错误。
- 商品已被删除。
- 前台链接还是旧地址。

处理方式：

- 在商品管理中确认商品存在。
- 检查 slug 是否与前台地址一致。
- 重新从商品列表点击进入详情页。

### 11.3 视频无法播放

可能原因：

- 视频链接错误。
- 外部视频平台限制播放。
- 网络无法访问视频平台。

处理方式：

- 在后台检查视频 URL。
- 使用浏览器单独打开视频链接验证。
- 更换为可嵌入播放的视频链接。

### 11.4 图片不显示

可能原因：

- 图片地址错误。
- 图片资源被删除。
- 图片链接不允许外部访问。

处理方式：

- 检查图片 URL。
- 使用浏览器单独打开图片地址。
- 更换为稳定可访问的图片地址。

### 11.5 无法登录后台

可能原因：

- 账号错误。
- 密码错误。
- 角色选择错误。
- 后端服务异常。
- 多次失败触发登录限制。

处理方式：

- 检查账号、密码和角色。
- 稍后再试。
- 联系管理员重置密码。

### 11.6 订阅邮件没有发送

可能原因：

- 邮件配置不正确。
- SMTP 服务不可用。
- 邮箱账号或密码错误。
- 邮件服务商限制发送。

处理方式：

- 检查邮件配置。
- 确认 SMTP 信息正确。
- 查看订阅记录中的邮件状态或错误提示。

## 12. 后台使用规范

### 12.1 内容规范

- 标题简洁明确。
- 摘要面向客户阅读。
- 图片清晰，避免变形。
- slug 使用英文小写、数字和短横线。
- 删除内容前先确认前台是否仍在使用。

### 12.2 商品维护规范

- 商品标题应与实际商品一致。
- SKU 保持唯一。
- 商品分类选择准确。
- 价格和库存及时更新。
- 精选商品数量不宜过多。

### 12.3 博客维护规范

- 发布前检查标题、分类、摘要、正文和封面图。
- 正文建议使用小标题分段。
- 不要发布空内容或测试内容。
- 删除文章前确认是否已有外部推广链接。

### 12.4 线索处理规范

- 新询盘应及时跟进。
- 状态变化后及时保存。
- 负责人填写清楚。
- 客户邮箱和联系信息不得随意外传。

### 12.5 账号安全规范

- 不共用管理员账号。
- 密码不要告诉无关人员。
- 离开电脑前退出后台。
- 恢复密钥只由管理员保管。
- 不在公共电脑上保存后台密码。

## 13. 日常工作流程建议

### 13.1 上架新商品

1. 先确认商品分类是否已存在。
2. 在商品管理中新建商品。
3. 填写商品标题、slug、SKU、分类、价格、图片、摘要和描述。
4. 设置状态和是否精选。
5. 保存。
6. 到前台商品列表和详情页检查。
7. 如需首页展示，到首页产品模块中选择该商品。

### 13.2 发布新文章

1. 确认文章分类是否存在。
2. 进入文章列表新建文章。
3. 填写标题、slug、分类、作者、发布时间。
4. 填写摘要、封面和正文。
5. 保存并发布。
6. 到前台博客列表和详情页检查。

### 13.3 更新首页内容

1. 修改首页轮播、产品、视频、分类或评价内容。
2. 检查首页模块开关。
3. 保存。
4. 刷新前台首页查看效果。
5. 在手机端和电脑端各检查一次。

### 13.4 处理客户询盘

1. 进入询盘管理。
2. 查看 new 状态询盘。
3. 根据客户留言联系客户。
4. 修改状态为 processing。
5. 跟进完成后修改为 done。
6. 无效或无需处理的询盘可归档。
